## Запуск приложения

### Установка всех пакетов для работы приложения (установить в двух папках backend и frontend)
```
npm install
```
### Запустить backend
```
node index.js
```

### Запустить frontend
```
npm run serve
```

### Запустить БД
```
MySQL
```
